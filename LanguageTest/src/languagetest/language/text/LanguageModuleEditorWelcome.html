Welcome to the Language Module Editor

  1. Choose a module:
    • To edit an existing module, open the module using the File menu or click on its name on the left hand side, if it is already open.
    • To create a new module choose New->Module from the File menu.
  2. Click the New Test Item button or select an existing Test Item to edit it. If you cannot see any Test Items in the module, click just to the left of the module name to expand the tree.
    • Enter the word/phrase in your own language
    • Enter the word/phrase in the language you are learning
    • Click to unlock the microphone and record your language helper or browse to an exising recording
    • If you have a picture, you can also add that to the Test Item
  3. You can also import Test Items from a CSV file:
    • Select the module that you want to import the items into
    • Use File->Import from the menu
    The CSV files should have 4 columns:
    "Native language","Foreign Language","Audio filename","Picture filename"
